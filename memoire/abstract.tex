\begin{abstract}
\hskip7mm

\begin{spacing}{1.3}

  Ce projet porte sur l'analyse de parties d'un jeu de cartes appelé \textit{Dominion} enregistrées sous forme de \textit{Logs}. L'objectif principal est d'en interpréter les données pour les rendre utilisables facilement afin de pouvoir effectuer des vérifications et des interprétations sur la base de celles-ci. Pour cela une première partie du projet porte sur le \textit{Parsing} de ces \textit{Logs}, puis dans un second temps le stockage des données extraites. La seconde partie du programme est constitué d'une bibliothèque capable de récupérer ces données, de les analyser et/ou de les reconstituer pour les compléter.

\end{spacing}
\end{abstract}
