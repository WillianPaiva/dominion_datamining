\begin{abstract}
\hskip7mm

\begin{spacing}{1.3}

  Ce projet porte sur l'analyse de parties sous formes de logs. L'objectif principal est d'obtenir des données utilisables facilement afin de pouvoir effectuer des analyses sur celles-ci. Pour cela une première partie du projet porte sur la lecture de ces logs puis dans un second temps le stockage des données extraites. La seconde partie du programme porte sur une bibliothèque capable de récupérer ces données , de les analyser et de les modifier.

\end{spacing}
\end{abstract}
